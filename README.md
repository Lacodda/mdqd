## Description
mqd is command line tool for quickly creating static documentation from the markdown files

## Installation

You can install the package from npm

```bash
npm i -g mqd
```

## Usage

```bash
mqd convert path/to/source/file.md path/to/destination/directory
```

## Tests

```bash
npm test
```
